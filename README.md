# BeatX

## Sistema Web de Atención Postventa

Proyecto académico desarrollado para la empresa BeatX.

El sistema automatiza:

- Registro de solicitudes postventa.
- Validación de datos.
- Asignación de técnicos.
- Gestión mediante Git y GitHub.
- Integración Continua (CI).
- Despliegue Continuo (CD).

Desarrollado para la Universidad Tecnológica del Perú.

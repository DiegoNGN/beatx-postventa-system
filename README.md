# Sistema Automatizado de Gestión de Atención al Cliente Post-Venta para BeatX

## Descripción

BeatX es una empresa dedicada a la comercialización de instrumentos musicales y accesorios. Actualmente presenta dificultades en la gestión de solicitudes post-venta debido al uso de herramientas manuales como hojas de cálculo.

Este proyecto tiene como finalidad desarrollar un sistema web que permita automatizar el registro, seguimiento y control de solicitudes post-venta, optimizando los tiempos de respuesta y mejorando la experiencia del cliente.

## Objetivo General

Desarrollar un sistema automatizado de gestión de atención al cliente post-venta para BeatX.

## Funcionalidades Principales

- Registro de solicitudes post-venta
- Validación de comprobantes de compra
- Verificación de autenticidad de productos
- Asignación automática de técnicos
- Seguimiento de solicitudes
- Generación de reportes
